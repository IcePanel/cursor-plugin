# IcePanel Cursor Plugin

A [Cursor](https://cursor.com) plugin that provides access to IcePanel, enabling AI assistants to manage models, connections, and more.

## Overview

This plugin connects your AI assistant to IcePanel.

## Installation

1. Open Cursor settings
2. Navigate to **Plugins**
3. Click **Browse Marketplace**
4. Search for "IcePanel"
5. Click **Install**
